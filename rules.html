Règles

GeoGames est un jeu de réflexion où vous devez réussir à deviner dans quel univers de jeu vidéo où vous avez atteri.

Vous avez le droit de regarder autour de vous et de zoomer pour observer l’environnement et les détails du décors mais il vous est impossible de vous déplacer.

Vous devez d’abord deviner dans quel jeu vous atterir pour gagner des points. Vous avez droit à 3 tentatives pour deviner le jeu.

Si vous avez trouvez le jeu, une carte du jeu apparaittra. À vous maintenant de placer sur la carte l’endroit exact où vous avez attéri. Plus vous êtes proche du point initial, plus vous marquerez des points
